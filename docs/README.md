# Docs Overview

If you want to set up the environment and run this project, start here:

`Setup_Guide.md`

That file contains:

- virtual environment setup
- dependency installation
- Playwright browser installation
- `.env` configuration
- run commands
- output file locations
- prompt log notes
- troubleshooting

Suggested order:

1. Read `Setup_Guide.md`
2. Configure `.env`
3. Run `python app.py <url>`
4. Review `output/latest.json`, `output/latest.txt`, and `prompt_logs/run.md`

The root `README.md` explains the product, architecture, AI design choices, trade-offs, and submission context. `Setup_Guide.md` is the main operational document for getting the tool running locally.
